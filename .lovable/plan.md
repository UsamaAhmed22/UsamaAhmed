

# DevOps Engineer Portfolio — Usama Ahmed

A modern, single-page portfolio website with a dark/light theme, smooth animations, and a professional DevOps-focused design.

## Design & Theme
- **Dark mode by default** with light mode toggle, using deep navy/black backgrounds with blue accent highlights
- Clean, minimal layout with generous spacing and modern sans-serif typography
- Subtle DevOps-themed iconography throughout (cloud, containers, pipelines)
- Smooth scroll navigation and fade-in/slide-in animations on scroll

## Sections

### 1. Sticky Navbar
- Navigation links to all sections with smooth scrolling
- Dark/light mode toggle
- "Download Resume" CTA button
- Mobile hamburger menu

### 2. Hero Section
- Bold headline with your DevOps specializations
- Professional summary as subheadline
- "View Projects" and "Contact Me" action buttons
- Animated tech badges (Jenkins, Docker, Kubernetes, GitOps, AWS, Azure, Linux)
- Subtle background animation or gradient

### 3. About Me
- Professional bio highlighting your CS degree from MAJU, Beenco Lab internship, and hands-on pipeline experience
- Clean card layout with a professional feel

### 4. Skills Grid
- Categorized skill cards with icons for each category: CI/CD, Containerization, Kubernetes, GitOps, Cloud, Security, Code Quality, OS & Scripting, Tools, and Processes
- Visual skill presentation with relevant tech icons

### 5. Projects Showcase
- 3 detailed project cards with descriptions and tech stack tags
- CI/CD & GitOps Pipeline on Azure, Full-Stack App with Docker & Compose, AWS Cloud Architecture Design
- "GitHub Repo" and "Case Study" buttons per project

### 6. Experience Timeline
- Vertical timeline layout showing your DevOps Intern role at Beenco Lab
- Key responsibilities listed with clean formatting

### 7. Certifications
- Badge-style cards for your 4 certifications from Coursera, KodeKloud, and CertiProf

### 8. Tech Stack Logos
- Visual logo/icon bar showing all your key technologies

### 9. Contact Section
- Contact info (email, location, LinkedIn, GitHub)
- Working contact form with Name, Email, Message fields and "Send Message" button
- Input validation

### 10. Footer
- Copyright notice, social icons for LinkedIn and GitHub
- "Back to Top" floating button

## Key Features
- Fully mobile responsive
- Dark/light mode toggle
- Smooth scroll navigation
- Scroll-triggered animations (fade-in, slide-in)
